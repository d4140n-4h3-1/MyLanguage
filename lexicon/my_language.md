## Word Formation Patterns

### Tables

| Singular | Plural |
|----------|--------|
| -d       | -ds    |
| -k       | -x     |
| -l       | -lum   |
| -r       | -rum   |
| -os      | -ei    |
| -us      | -ei    |

### Plural Suffix: *-um*
Used for all nouns:
- *paþirum* (fathers), *bernum* (children), *kronum* (times)

### Abstract Noun Suffix: *-tun*
Forms abstract nouns from verb roots:
- *evolutun* (evolution), *edukatun* (education), *mutatun* (mutation)

### Adjective Suffix: *-lik*
Forms adjectives from nouns:
- *biosik* (alive/living), *anyarlik* (yearly), *viklik* (weekly)

### Verb Pattern: *-son*
Present tense conjugation incorporates the pronoun:
- *mison* (I am), *deuson* (you are), *vison* (he/she/it is)
- *mirson* (we are), *dirson* (you are), *virson* (they are)

### Present and Past Tense Suffixes

| Function | Word   | Usage                                                     |
|----------|--------|-----------------------------------------------------------|
| -ed      | *-et*  |                                                           |
| -ed      | -n     | If the last letter of a word is a vowel                   |
| -ing     | *-end* |                                                           |
| -s       | *-s*   |                                                           |
| -        | -el    | Future tense. Non-existant in any language I am aware of. |

## Time

### Basic Time Words

| English              | Word         | Plural    |
|----------------------|--------------|-----------|
| time                 | *kron*       | *kronum*  |
| all times / eternity | *omnikronum* | -         |
| minute               | *minet*      | *minetum* |
| week                 | *vik*        | *vikum*   |
| month                | *manoþ*      | *manoþum* |
| year                 | *anyar*      | *anyarum* |

"Anyar" comes from "annual" and inspired by "año", and the "yar" part comes from "jar".
